OnSSET

Open Source Spatial Electrification Tool

OnSSET is a GIS-based model for electricity access planning in developing countries. Each Starter Data Kit contains geospatial and socio-economic input files ready for use directly in OnSSET, grouped into three categories.

Large files (>25 MB) will show a Google virus-scan warning — click Download anyway to proceed.
