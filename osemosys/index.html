About

OSeMOSYS is an open-source framework for long-run energy planning. Each Starter Data Kit contains pre-assembled input data to help you build a country-level energy system model.

📋 What’s included

Country-specific energy supply and demand data formatted for use with the OSeMOSYS modelling framework. Ready to use as a starting point for energy system analysis.
